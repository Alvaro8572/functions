# .functions

🎯 Una aplicación web con herramientas útiles de matemáticas, nutrición y productividad.

![Estado](https://img.shields.io/badge/status-activo-brightgreen)
![Licencia](https://img.shields.io/badge/licencia-MIT-blue)

## 🚀 Características

### 📐 Herramientas Matemáticas
- **Triangle 180°** - Genera 3 ángulos de triángulo válidos
- **Fibonacci** - Calcula la secuencia de Fibonacci
- **Polygons** - Busca nombres de polígonos (3-20 lados)
- **Binomial Generator** - Genera expresiones binomiales (a ± b)²
- **Polynomial Generator** - Genera expresiones polinómicas
- **Unit Converter** - Convierte unidades de peso, longitud y temperatura

### 🥗 Herramientas de Nutrición
- **BMI Calculator** - Calcula tu Índice de Masa Corporal

### ⚡ Herramientas de Productividad
- **Secure PIN Generator** - Genera PINs numéricos seguros
- **Word Generator** - Descubre palabras interesantes en inglés
- **Password Generator** - Genera contraseñas seguras

### 🧪 Daily Lab
Sistema de misiones diarias para gamificar el uso de la aplicación.

## 🛠️ Tecnologías

- HTML5
- CSS3 (Tailwind CSS + CSS personalizado)
- JavaScript (ES6+)
- LocalStorage para persistencia

## 📦 Instalación

1. **Clonar el repositorio:**
```bash
git clone https://github.com/Alvaro8572/.functions.git
```

2. **Abrir en el navegador:**
 Simplemente abre el archivo `index.html` en tu navegador favorito.

No necesita servidor web ni instalación adicional.

## 🚀 Uso

1. Abre `index.html` en tu navegador
2. Selecciona una categoría (Matemáticas, Nutrición o Productividad)
3. Usa las herramientas disponibles
4. Completa las misiones diarias en el panel "Daily Lab"
